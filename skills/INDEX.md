# Plugin Skills Index

## 使用顺序（默认）
1. 先按任务类型在下表里选最匹配的 skill。
2. 只读取命中的 `SKILL.md`，不要全量加载。
3. 若需多 skill，按“主技能 -> 校验技能”组合。

## Skills 列表

1. `siyuan-block-edit-safe`
- 用途：块级安全编辑、回读校验、失败回滚。
- 适用：改块内容、插入块、批量修正文案。
- 文件：`skills/siyuan-block-edit-safe/SKILL.md`

2. `siyuan-image-rules`
- 用途：图片导入/截图、标题与说明写入、读回验证。
- 适用：帖子配图、网页截图入笔记、图片说明补齐。
- 文件：`skills/siyuan-image-rules/SKILL.md`

3. `siyuan-research-notes`
- 用途：双来源交叉验证 + 可复现调研记录。
- 适用：查资料、做对比、给结论并附证据。
- 文件：`skills/siyuan-research-notes/SKILL.md`

4. `siyuan-docs-cleanup`
- 用途：文档去重、合并、归档并保留溯源。
- 适用：README 整理、重复文档治理、资料归档。
- 文件：`skills/siyuan-docs-cleanup/SKILL.md`

5. `siyuan-meeting-actions`
- 用途：会议纪要结构化整理与行动项拆解。
- 适用：会后纪要、决策确认、任务分配追踪。
- 文件：`skills/siyuan-meeting-actions/SKILL.md`

6. `worker`
- 用途：`$team` worker 生命周期（ACK/claim/execute/update）。
- 适用：tmux 团队并行执行场景。
- 文件：`skills/worker/SKILL.md`
